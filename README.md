# DecodeBot — Data Classification Using AI

Project 2 of the Decode Labs. A K-Nearest Neighbors (KNN) classifier built with Python and scikit-learn using the Iris dataset.

## What it does

Loads the Iris dataset, splits it into training/testing sets, trains a KNN classifier, evaluates accuracy, and classifies custom flower measurements.

## Quick start

```bash
pip install scikit-learn
python classifier.py
```

## Project structure

```
classifier.py           Entry point. KNN training, evaluation, and prediction.
```
